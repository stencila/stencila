Welcome to Stencila

Stencila is the office suite for reproducible research. With Stencila you can produce rich computation-intensive publications without requiring programming knowledge.

Cells are the excutable (dynamic) bits of a document. They produce a result as you type. Try to modify the following cell:

Cells use a very simple expression language called Mini to allow you to perform common data analysis and visualisation tasks within the document.

You can edit this document just like in a traditional word processor. Press enter to start a new paragraph or insert a table or cell.

Cells can also be used to plot your data. The plot function is a generic function that allows you to visualize data using various visual encodings.

The nice thing about using Mini is that the dependency between cells can be automatically determined. So if you change the value in one cell, all the other cells that depend on tht value will automatically update - just like in a spreadsheet. Try altering the filtering expression in the filtered_data cell above.

You can also let your readers alter document variables directly using inputs and inspect their value with outputs. Heres an example: if the diameter of a circle is m then it's circumference is 213.52 m.

See the Stencila Manual for more on what you can do with Mini, including running code in other languages.
