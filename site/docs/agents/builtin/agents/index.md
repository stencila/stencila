Agents for creating and reviewing other agents.

You can use these agents in the Stencila TUI by selecting one with the `/agent` command, or by starting your prompt with `#agent-name` e.g. `#agent-creator`. Agents can also be referenced by name in workflow node definitions.

- [**Agent Creator Agent**](./agent-creator/): Creates or updates an agent
- [**Agent Reviewer Agent**](./agent-reviewer/): Reviews an agent for quality, correctness, and completeness
