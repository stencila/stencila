General-purpose agents handle a wide range of tasks and route requests to specialized agents when appropriate.

- [**Manager**](./manager/): Routes user requests to the most appropriate agent or workflow.
- [**General**](./general/): A general-purpose agent using the default model provider
- [**General A**](./general-a/): A general-purpose agent using Anthropic's frontier model
- [**General G**](./general-g/): A general-purpose agent using Google's frontier model
- [**General M**](./general-m/): A general-purpose agent using Mistral's frontier model
- [**General O**](./general-o/): A general-purpose agent using OpenAI's frontier model
