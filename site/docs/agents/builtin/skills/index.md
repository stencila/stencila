Agents for creating and reviewing agent skills.

> [!tip] Usage
>
> You can use these agents in the Stencila TUI by selecting one with the `/agent` command, or by starting your prompt with `#agent-name` e.g. `#skill-creator`. Agents can also be referenced by name in workflow node definitions.

- [**Skill Creator Agent**](./skill-creator/): Creates or updates a skill
- [**Skill Reviewer Agent**](./skill-reviewer/): Reviews a skill for quality, correctness, and completeness
