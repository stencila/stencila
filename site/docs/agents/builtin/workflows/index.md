Agents for creating and reviewing workflows.

You can use these agents in the Stencila TUI by selecting one with the `/agent` command, or by starting your prompt with `#agent-name` e.g. `#workflow-creator`. Agents can also be referenced by name in workflow node definitions.

- [**Workflow Creator Agent**](./workflow-creator/): Creates or updates a workflow
- [**Workflow Reviewer Agent**](./workflow-reviewer/): Reviews workflows for correctness, clarity, and completeness
