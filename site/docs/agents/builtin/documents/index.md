Agents for creating and reviewing documents.

> [!tip] Usage
>
> You can use these agents in the Stencila TUI by selecting one with the `/agent` command, or by starting your prompt with `#agent-name` e.g. `#figure-creator`. Agents can also be referenced by name in workflow node definitions.

- [**Figure Creator Agent**](./figure-creator/): Creates or updates figures in Stencila Markdown documents, including simple image figures, executable figures, multi-panel layouts, captions, and SVG annotation overlays
- [**Figure Reviewer**](./figure-reviewer/): Reviews Stencila figures using the figure-review skill, focusing on structure, captions, overlays, rendering risks, scientific annotation safety, and approval readiness.
