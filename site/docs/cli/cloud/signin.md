Sign in to Stencila Cloud

# Usage

```sh
stencila cloud signin [OPTIONS]
```

# Examples

```bash
# Sign in to Stencila Cloud
stencila cloud signin

# Sign in manually
stencila cloud signin --manual

# Use one of the command aliases
stencila signin
stencila login
```

# Options

| Name           | Description                                                                            |
| -------------- | -------------------------------------------------------------------------------------- |
| `-m, --manual` | Signin by manually entering a Stencila access token. Possible values: `true`, `false`. |
