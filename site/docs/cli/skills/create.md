Create a new skill

Creates a new skill directory with a template SKILL.md in the current workspace's `.stencila/skills/` directory.

# Usage

```sh
stencila skills create <NAME>
```

# Examples

```bash
# Create a new skill
stencila skills create my-new-skill
```

# Arguments

| Name     | Description                 |
| -------- | --------------------------- |
| `<NAME>` | The name for the new skill. |
