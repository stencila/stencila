Remove a route

# Usage

```sh
stencila site remove <ROUTE>
```

# Examples

```bash
# Remove a route
stencila site remove /about/
stencila site remove /old/
```

# Arguments

| Name      | Description                             |
| --------- | --------------------------------------- |
| `<ROUTE>` | Route path to remove (e.g., "/about/"). |
