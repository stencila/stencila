Remove an output configuration

# Usage

```sh
stencila outputs remove <OUTPUT>
```

# Examples

```bash
# Remove an output
stencila outputs remove report.pdf

# Remove a spread output
stencila outputs remove "{region}/report.pdf"
```

# Arguments

| Name       | Description       |
| ---------- | ----------------- |
| `<OUTPUT>` | Output to remove. |
