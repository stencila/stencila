Manage linters

# Usage

```sh
stencila linters [COMMAND]
```

# Examples

```bash
# List all available linters
stencila linters

# Lint a file using a linter
stencila linters lint script.py
```

# Subcommands

| Command           | Description                |
| ----------------- | -------------------------- |
| [`list`](list.md) | List the linters available |
| [`lint`](lint.md) | Lint a file                |
