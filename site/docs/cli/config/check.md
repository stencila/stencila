Check configuration validity

# Usage

```sh
stencila config check [DIR]
```

# Examples

```bash
# Check config in current directory
stencila config check

# Check config in a specific directory
stencila config check ./my-project
```

# Arguments

| Name    | Description                                               |
| ------- | --------------------------------------------------------- |
| `[DIR]` | Directory to check configuration for. Default value: `.`. |
