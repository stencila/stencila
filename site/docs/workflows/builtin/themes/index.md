Workflows for creating and refining Stencila themes.

You can run these workflows in the Stencila TUI by selecting one with the `/workflow` command, or by starting your prompt with `~workflow-name` e.g. `~theme-creation-iterative`.

- [**Theme Creation Iterative Workflow**](./theme-creation-iterative/): Create and iteratively refine a Stencila theme using the `theme-creator` and `theme-reviewer` agents, route approved drafts through human review with optional commit, and loop on requested revisions
