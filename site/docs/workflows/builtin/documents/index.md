Workflows for creating and refining documents.

> [!tip] Usage
>
> You can run these workflows in the Stencila TUI by selecting one with the `/workflow` command, or by starting your prompt with `~workflow-name` e.g. `~figure-creation-iterative`.

- [**Figure Creation Iterative Workflow**](./figure-creation-iterative/): Create and iteratively refine a Stencila figure using the `figure-creator` and `figure-reviewer` agents, route approved drafts through human review with optional commit, and loop on requested revisions
