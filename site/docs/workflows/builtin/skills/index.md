Workflows for creating and refining agent skills.

You can run these workflows in the Stencila TUI by selecting one with the `/workflow` command, or by starting your prompt with `~workflow-name` e.g. `~skill-creation-iterative`.

- [**Skill Creation Iterative Workflow**](./skill-creation-iterative/): Create and iteratively refine a Stencila skill using the `skill-creator` and `skill-reviewer` agents, with human review, optional commit, and revision loops until accepted
