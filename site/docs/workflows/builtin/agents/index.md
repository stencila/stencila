Workflows for creating and refining agents.

- [**Agent Creation Iterative**](./agent-creation-iterative/): Create and iteratively refine a Stencila agent using the agent-creator and agent-reviewer agents, route approved drafts through human review with optional commit, and loop on requested revisions
- [**Agent Creation Topdown**](./agent-creation-topdown/): Design an agent top-down by first planning its skills, creating each skill via the skill-creation-iterative workflow, and then creating, refining, and optionally committing the agent with those skills available
